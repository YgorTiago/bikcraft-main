Este projeto foi desenvolvido durante o curso da Origamid com o objetivo de aplicar, na prática, conhecimentos de HTML, CSS e JavaScript na construção de um site responsivo para uma marca fictícia de bicicletas.

Tecnologias utilizadas:

HTML5

CSS3

JavaScript

Objetivo:
Consolidar conceitos fundamentais de desenvolvimento web, incluindo criação de layouts, responsividade, manipulação do DOM e organização de arquivos em um projeto realista para fins educacionais.
*Projeto desenvolvido para fins acadêmicos no curso Origamid.
